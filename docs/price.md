# Prices

Official API Documentation: https://developer.walmartapis.com/#prices

## Update a single Item Price

```php
<?php
use Walmart\Price;

$client = new Price([
    'consumerId' => getenv('CONSUMER_ID'),
    'privateKey' => getenv('PRIVATE_KEY'),
]);

$price = $client->update([
    'sku' => 'ABC123', // required
    'currency' => 'USD', // required
    'price' => '14.99', // required
]);
```
